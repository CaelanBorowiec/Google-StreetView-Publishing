Post-processing

Install Software

If you are using a pre-configured computer, you may skip this section. Otherwise you will need to install the following software:

  • Kolor Autopano Giga
  • pyTiler †

† pyTiler does not require installation. Just download and extract the folder to a convenient location.

Copy & Group Images

Copy the images from the camera to your computer. Divide the photos into folders for each viewpoint set.

Important: Give each folder a name that corresponds with the viewpoint. This name will be used to specify viewpoint names and may be publicly visible, choose wisely!
The name should not contain spaces or special characters Be logical and/or descriptive with your naming conventions. Example: pump_room_1st-floor.

Stitching Viewpoint(s)

 

  1. Open Kolor, then drag and drop folders containing unprocessed image sets to the left pane. [image]
  2. Click the green Detect All button and wait for Kolor to complete pre-processing of all selected image sets.
  3. Take a look at the preview pictures and check for parallax errors. If everything looks good, click the Render All button to open the render settings menu.
  4. Select desired rendering settings. For best results, choose the following settings:
    • Output size: Max / 100%
    • Interpolator: Bicubic
    • Blending preset: Anti-ghost
    • Format: Jpeg / 8Bit / 9) High / 72DPI
    • Exported data: Panorama
  5. Select a folder to save the stitched images to and click "Render".

Kolor will open a Batch Render window and begin to process the images. This may take some time. Wait for the stitching process to complete, them move on to Creating Tiles.

Creating Tiles

 

  1. Run pyTiler.exe. It can be found in the pyTiler-1.2.0-x64 folder after you extract it.
  2. In the popup, browse to the folder containing your named, stitched, panorama images
  3. Choose a processing option by entering 1 (automatic, highest-quality), 2 (prompt for each image), or 3 (same settings for all images). Unless you are debugging or have special needs, you should always choose option 1.
  4. Hit enter so the program will generate HTML/JavaScript data for each viewpoint
  5. Confirm that the images have correct viewpoint names. You will not be able to easily update names after this step.
  6. Hit Enter one more time to begin batch processing.

After pyTiler has finished, proceed to the image folder you selected for processing. Inside you should see a folder named "tiles". Continue to the Web Software section for information on what to do next.
